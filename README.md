# Restaurant Chatbot Miami

An AI chatbot for restaurants in Miami that handles customer questions, provides hours, and manages reservations — automatically.

## What it does

- Answers customer questions in real time
- Provides restaurant hours and menu information
- Handles reservation requests
- Works in English and Spanish
- Available 24/7 with no staff required

## Built with

- JavaScript
- Node.js
- OpenAI API

## Who is this for

Any Miami restaurant that wants to reduce time answering repetitive messages and never miss a reservation request again.
